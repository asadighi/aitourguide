# architecture

This folder contains technical architecture documentation for AI Tour Guide.

Architecture files guide implementation but do not override `/spec`.
If there is a conflict, `/spec` overrides `/architecture`.

## files

- `architecture-overview.md` — system diagram, package boundaries, request flows
- `tech-stack.md` — chosen stack with rationale
- `data-persistence.md` — entity model, invariants, migrations
- `auth-and-security.md` — auth model, sessions, secrets
- `deployment-hosting.md` — environments, deployment, config
- `llm-integration.md` — provider, prompts, cost controls
- `mcp-architecture.md` — MCP tools, contracts, prompt management
